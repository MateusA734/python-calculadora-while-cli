# Python Calculadora CLI com While

Este projeto foi desenvolvido como exercício de lógica de programação utilizando Python.

A aplicação consiste em uma calculadora simples executada no terminal (CLI), capaz de realizar operações matemáticas básicas entre dois números digitados pelo usuário.

O programa permanece em execução através de um loop `while`, permitindo que vários cálculos sejam realizados até que o usuário escolha encerrar o programa.

---

## Funcionalidades

A calculadora permite realizar as seguintes operações matemáticas:

- Adição (+)
- Subtração (-)
- Multiplicação (*)
- Divisão (/)

O programa solicita ao usuário:

1. O primeiro número
2. O operador matemático
3. O segundo número

Após realizar o cálculo, o resultado é exibido no terminal.

Em seguida, o programa pergunta ao usuário se deseja encerrar ou continuar realizando novos cálculos.

---

## Conceitos praticados

Durante o desenvolvimento deste exercício foram utilizados conceitos fundamentais da linguagem Python:

- Estrutura de repetição `while`
- Entrada de dados com `input()`
- Conversão de tipos com `float()`
- Estruturas condicionais (`if`, `elif`, `else`)
- Tratamento de exceções (`try` / `except`)
- Validação de operadores matemáticos
- Controle de fluxo com `continue` e `break`
